Blinkenpad

mini remake of blinkenlights, which will be perhaps at congress

Download as .zip Download as .tar.gz View on GitHub

Are you sad Blinkenlights is over?

Well, I got something for ya!

I will use a Novation Launchpad - a MIDI controller, but there are LEDs under the buttons, which create a nice matrix(8x8). And YOU can participate!

All you have to do is to write animations(unfortunately no VE yet) in a python array. It looks like this:

[
 [
  [red leds], [yellow leds], [green leds]
 ]
 [
  [red leds], [yellow leds], [green leds]
 ]
 [
  [red leds], [yellow leds], [green leds]
 ]
 [
  [red leds], [yellow leds], [green leds]
 ]
 [
  [red leds], [yellow leds], [green leds]
 ]
]

Or, to explain it better:

[(animation start)
 [(frame start)
  [(all red leds in this frame)], 
  [(all yellow leds in this frame)], 
  [(all green leds in this frame)]
 (frame end)]
(animation end)]

(don't mind the round brackets)
